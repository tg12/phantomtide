# Phantom Tide

**Global Maritime Intelligence Platform**

---

The ocean is not transparent.

Phantom Tide is a live intelligence platform for maritime analysts who need to know not just where things are — but where they aren't, and why.

This repository is the public interface for feedback and access requests. The platform itself is private.

---

## Access

Phantom Tide is not publicly available. If you believe you have a use case, open an issue or contact directly.
